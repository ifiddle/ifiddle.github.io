1. 如要安裝 MCO 播放器，電腦系統需要什樣的條件？
 

基本系統要求

作業系統： - Microsoft Windows 98 SE
- Windows ME
- Windows XP
- Windows Vista
硬件： - 音效卡
- 128 MB 以上記憶體
- 100 MB 以上硬碟空間
軟件： - Microsoft Windows Media Player 9 或以上版本
- Microsoft .net Framework 2.0 或以上版本 (安裝程式會自動為您安裝)
網絡連線： - 14.4kbps以上的速度網絡連線或透過寬頻網絡, 3G , GPRS分組數據連線

2. 我可以使用 MCO 播放器播放 vcd, dvd 及 mp3 的檔案嗎？
  不可以。MCO 是一個嶄新的網上電視頻道，並會根據您所選擇的頻道內容，下載及播放相關最新資訊。
> Top
3. 頻道內容的影片，是否只播放一次？我可以重複觀看嗎？
 

播放器會自動保留您的最新瀏覽記錄，最高可保留 25 個記錄。MCO 播放器提供兩種途徑，方便用戶重複觀賞頻道內容：

  1. 點?MCO 播放器右上角 “瀏覽記錄” 的按鈕，列出已觀賞的頻道影片名稱，單?相關名稱，播放器便會自動播放相關內容。
  2. 在 MCO 播放器的選項視窗內，選擇 “記錄”。單?相關名稱，播放器便會自動播放相關內容。

> Top
4. 我可以離線觀賞頻道內容嗎？
  可以。MCO會在您的個人電腦閒置及在線時，自動下載新的內容。您可以隨時離線觀看已載入MCO播放器的內容。
> Top
5. 可否預設自動播放新片？
  可以。在 MCO 播放器的選項視窗內，選擇 “一般”。在 “內容更新提示” 一項，選取 “自動播放” 便可。
> Top
6. 我可以把MCO的片段上傳到自己的 PDA 、手提電話上播放嗎？
  目前，MCO的內容只能夠於MCO上播放。但我們的開發團隊正預備為MCO推出 PDA 版本，不久將來更會開發其它平台上的MCO 軟件。
> Top
7. 我需要重新安裝/移除MCO，如何保留我於「MCO服務中心」裡的積分？
  在「MCO服務中心」內「說明」頁面底部，紀錄了一組支援號碼。
請於重新安裝/移除MCO軟件前，自行記錄這組支援號碼。並於新安裝後再把新的支援號碼連同舊有支援號碼與積分數目，傳送到支援電郵：support@mychannel.hk。 請注意，我們只能夠為最近30天內被移除的MCO軟件提供積分轉移服務。
> Top
8. 如何得知新片是否已經下載？
 

MCO 會於新片段已被下載時，MCO的圖示會閃動並會顯示氣球形狀的提示視窗通知您。

  1. 點?MCO 播放器右上角 “瀏覽記錄” 的按鈕，列出影片名稱，單?相關名稱，播放器便會自動播放相關內容。
  2. 在 MCO 播放器的選項視窗內，選擇 “記錄”。將會列出所有已載入MCO的影片。 ，單?相關名稱，播放器便會自動播放相關內容。

> Top
9. 為什麼未能下載或播出最新片段？
  為節省個人電腦及網絡資源。MCO只會於所有「記錄」中的影片狀態為「已觀看」及 電腦能夠連接互聯網時下載內容。
當然，亦有可能是您於「個人選擇」-> 「喜好」中選擇的所有頻道暫時未有新內容。您可以嘗試於「喜好」中選取更多的頻道。
> Top
10. 我無法顯示「MCO服務中心」的遊戲？
 

MCO的部份功能需要使用 Flash Player，請到 Adobe 網站下載最新版本的 Flash 播放軟件。

  為節省個人電腦及網絡資源。MCO只會於所有「記錄」中的影片狀態為「已觀看」及 電腦能夠連接互聯網時下載內容。
當然，亦有可能是您於「個人選擇」-> 「喜好」中選擇的所有頻道暫時未有新內容。您可以嘗試於「喜好」中選取更多的頻道。
> Top
12. 我在使用MCO軟件時遇到問題，您們會提供支援服務嗎？
  MCO設有支援電郵，您可能隨時與我們的服務團隊聯絡。
> Top
13. 當我於Windows Vista 於MCO的「個人選擇」按下「確認」或「套用」時，出現錯誤訊息。
  如您正在使用Windows Vista 或 Windows XP最新推出的 SP3 ，並啟用「使用者帳戶控制」 功能則可能出現這個錯誤訊息。請下載最新版本的MCO並重新安裝。
> Top
14. 什麼是Microsoft .net Framework ？
  Microsoft .net Framework是由Microsoft推出的最新應用程式框架，由Windows XP SP2起已被廣泛支援。使用.net Framework開發的程式具有更大的跨平台性能及高度安全性。
